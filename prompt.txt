Next.js로 아주 간단한 로그인 페이지를 만들어줘.

요구사항:

* Next.js 기반으로 구현
* App Router 사용
* TypeScript 사용
* Tailwind CSS를 사용해서 스타일링
* 로그인 페이지에는 아래 요소가 필요함

  * 이메일 입력창
  * 비밀번호 입력창
  * 로그인 버튼
* 전체 화면 중앙 정렬
* 반응형으로 구성
* 기본적인 유효성 검사 포함

  * 이메일 형식 체크
  * 비밀번호가 비어 있으면 에러 메시지 표시
* 로그인 버튼 클릭 시 실제 인증 대신 `console.log`로 입력값 출력
* 로딩 상태 처리
* 에러 메시지와 입력 상태가 사용자에게 보기 좋게 표시되도록 해줘

추가 요구사항:

* 가능한 한 컴포넌트를 깔끔하게 작성해줘
* 초보자가 이해하기 쉽게 코드에 간단한 주석 추가
* 파일 구조도 같이 설명해줘
* 필요한 경우 `app/login/page.tsx` 기준으로 작성해줘
* 바로 실행 가능한 코드로 만들어줘
* UI 디자인 작업은 **클로드 코드 플러그인 `frontend-design`** 을 적극 활용해줘
* 전체 UI는 **모던하고 미니멀한 스타일**로 구성해줘
* 카드형 레이아웃, 적절한 그림자, 라운드, 균형 잡힌 여백, 깔끔한 타이포그래피를 활용해서 세련된 로그인 화면으로 만들어줘
* 다만 과한 색상, 복잡한 장식, 불필요한 애니메이션은 피하고, 심플하면서도 완성도 높게 보여야 해

출력 형식:

1. 필요한 파일 목록
2. 각 파일의 전체 코드
3. 실행 방법
4. 코드 설명
